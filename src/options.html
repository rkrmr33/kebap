Keyboard shortcuts

Control+K
Open or close the feedback panel
HoldOption
Inspect elements under the pointer
↑↓
Move to the parent or child while inspecting
Enteror click
Select the highlighted element
Esc
Stop inspecting or cancel the current comment
Enter
Add the current comment to the queue
Shift+Enter
Insert a new line in a comment
Option+Shift+C
Copy the queue as Markdown
Option+Shift+X
Copy and clear the queue

The panel shortcut can be changed from the browser's extension keyboard-shortcuts page.
